# Agent Starter

A minimal React + Tauri template for building modern AI agent application.

## Project Setup

### Install

```bash
pnpm install

# For Android
pnpm tauri android init

# For ios
pnpm tauri ios init
```

### Development

```bash
# For Web
pnpm dev

# For Desktop
pnpm tauri dev

# For Android
pnpm tauri android dev

# For ios
pnpm tauri ios dev
```

### Build

```bash
# For Web
pnpm build

# For Desktop
pnpm tauri build

# For Android
pnpm tauri android build

# For ios
pnpm tauri ios build
```
